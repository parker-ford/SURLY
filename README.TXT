The main function is located in the main class. The program must be passed in a text file as a command line argument.
All .java files required are in the src folder.

Terminal commands:
	javav*.java
	
	java Main test.txt
	
	
Note: The printing was design with a fixed weight font in mind. The output is formatted correctly in terminal however with some IDEs, such as Intellij, the output is slightly off.
